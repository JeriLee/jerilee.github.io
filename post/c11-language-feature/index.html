18.5 包装器

std::function
便于泛型实例化，避免实例化很多次

例如：
对于泛型函数

template<typename F>
void use_f(F func) {
  static int count = 0;
  cout << (&count) << endl;
}

调用函数

use_f(my_add);                //1泛型实例
use_f(my_mul);                //1泛型实例
use_f(Fake1(2));              //2泛型实例
use_f(Fake1(4));              //2泛型实例
use_f(Fake2(8));              //3泛型实例
use_f(Fake2(16));             //3泛型实例
use_f([](double x) ->double {return x + 2;}); //4泛型实例
use_f([](double x) ->double {return x + 2;}); //5泛型实例

实例化函数和仿函数

double my_add(double x) {
  return x + x;
}

double my_mul(double x) {
  return x * 2;
}
struct Fake1 {
  int n_;
  Fake1(int n) :n_(n) {}
  bool operator()(int x) {
    return x + n_;
  }
};
struct Fake2 {
  int n_;
  Fake2(int n) :n_(n) {}
  bool operator()(int x) {
    return x + n_;
  }
};

** 解决办法：把所有函数包装成std::function<double(double)> 实例化即可改为一次
** 解决办法2：void use_f(T t, std::function<T(T)> func);

  show_list(args...);
}
