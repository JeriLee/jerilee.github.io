C#9.0 新内容

2020-09-21 3 min read

由于2020年9月还未发布正式版C# 9.0 所以本文内容暂时不全，后续会逐步完善

关于Try Catch Finally代码时序问题

try代码块中如果return，会将return的内容拷贝出来，再执行finally语句块。catch同理

C# 9.0 language feature

函数重写可以使用派生类返回值

public class Food { }
public class Meat : Food { }
public class Poke : Meat { }
public abstract class Animal {
  public abstract Food GetFood();
}

public class Tiger : Animal{
  public override Meat GetFood(){
    throw new NotImplementedException();
  }
}

public class BigTiger : Tiger{
  public override Poke GetFood(){
    throw new NotImplementedException();
  }
}

操作符 ?? 和 ?:

Person person = student ?? customer; // 共享基类，截止16.8.0Preview 3.0版本，还未实现该功能
int? result = b ? 0 : null; // 可null的值类型

new操作符省略类型

Point p = new (3, 5);

Switch匹配

传统代码

public static decimal CalculateToll(object vehicle) =>
    vehicle switch {
       ...
       
        DeliveryTruck t when t.GrossWeightClass > 5000 => 10.00m + 5.00m,
        DeliveryTruck t when t.GrossWeightClass < 3000 => 10.00m - 2.00m,
        DeliveryTruck _ => 10.00m,

        _ => throw new ArgumentException("Not a known vehicle type", nameof(vehicle))
    };

对于

DeliveryTruck _ => 10.00m,

可以省略为

DeliveryTruck => 10.00m,

新的书写方式

DeliveryTruck t when t.GrossWeightClass switch {
    > 5000 => 10.00m + 5.00m,
    < 3000 => 10.00m - 2.00m,
    _ => 10.00m,
},

加入新的or and not

>= 3000 and <= 5000 => 10.00m,

判断是否为空

not null => xxx,
null => xxx,

另外 对于语句

if (!(e is Customer)) { ... }

可以用新的方式书写

if (e is not Customer) { ... }

顶级语法

using System;

Console.WriteLine("Hello World!");

比较相等

问题：Equal 处理基类和派生类，会采用不同的重写
新的函数：EqualityContract

Record标识符 和 with标识符

public record A(int Id,string Name);
public static void main() {
  A a = new A(1, "");
  A b = a with { Id = 2 };
  Console.WriteLine(a.Id);
  Console.WriteLine(b.Id);
}

属性标记init

对于init标记的属性，只能在构造和初始化的时候修改。

public class Person
{
  public int Id { get; init; } = 2;
  public string Name;
  public Person(int a, string b)
  {
    Id = a;
    Name = b;
  }
}

反构造

A a = new A(1, ""); //构造
var (id, name) = a; //反向构造

record或class或struct都可以反向构造

public record A{
  public int Id;
  public string Name;
  public A(int id, string name)
    => (Id, Name) = (id, name);
  public void Deconstruct(out int id, out string name)
    => (id, name) = (Id, Name);
}
